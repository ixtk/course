შექმენით ახალი React-ის პროექტი: npm create vite@latest ბრძანებით

დაამატეთ 4 input ველი: title, description, tags და due date. Due date უნდა იყოს თარიღის (`Date`) ტიპის.

![Project form interface](/homework/descriptions/img-attachments/11-1.png)

არ ხართ შეზღუდული ამ კომპონენტის სტილში. შეგიძლიათ მთლიანი კოდი `App.jsx`-ში დაამატოთ.

1. Save ღილაკზე დაჭერისას, კონსოლში უნდა გამოჩნდეს შეყვანილი ინფორმაცია. არ არის აუცილებელი ვებ-გვერდზე გამოიტანოთ. ასეთი ფორმატით:

![Form submission output](/homework/descriptions/img-attachments/11-2.png)

2. Tag input-ში ტექსტის აკრეფისას, თუ მძიმეს ავკრეფთ, მძიმემდე რა ტექსტიც ეწერა უნდა დაემატოს tag-ების სიაში და input-ის ტექსტი წაიშალოს (ნახეთ მიბმული ვიდეო).

3. რომელიმე tag-ს თუ დავაჭერთ, უნდა წაიშალოს tag-ების სიიდან.

4. თუ იგივე tag-ის ტექსტს ჩავწერთ და მძიმის კლავიშს დავაჭერთ, არ უნდა დაემატოს სიაში.
